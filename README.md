# CG Lab 2: Kart Game

## Functionality

* Read mesh from OBJ file
* Render with Phong lighting model
* Manipulate mesh with mouse
* Translucent effect by OIT

See [lab report](doc/report.md) for details.

## Build

* CMake
* GCC 8.1.0 (or other compilers that support C++14)

## Dependency

* OpenGL 4.2
* GLFW 3.3
* GLEW 2.1
* Assimp 5.0
